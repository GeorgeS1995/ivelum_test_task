# Prepare venv

This project are using [poetry](https://python-poetry.org/) as dependency management tool

Prepare envs:  
`poetry install`  
Run shell:  
`poetry shell`

# Usign project

Run server:

`python main.py`

Run tests:

`pytest ./tests`
